This is my second public R Shiny APP on Github. The data is from Kaggle Fruit-360: https://www.kaggle.com/datasets/moltean/fruits

The CNN model was build from scratch, no transfer training is used. The Python code for the model can be found here: https://www.kaggle.com/code/pingliaobrizendine/fruitapp2python
************This model took more than 4 hours to train by using Kaggle GPU, free tier version. 

The app is deployed at: https://www.shinyapps.io/admin/#/application/16924005/logs
